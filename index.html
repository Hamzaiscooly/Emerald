Emerald Bot may collect and store the following information:

  • User Information: User IDs and usernames for functionality like moderation, leveling, and welcome messages
  • Message Content: Limited message content for moderation purposes (filtered for inappropriate content)
  • Command Usage: Data about commands used to improve our services

We use the collected information to:
